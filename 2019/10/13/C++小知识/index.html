一些初学者的笔记

NULL

NULL是在标准库头文件中定义的宏

它的值是一个空指针常量

C中常数0和（void*）0都是空指针常量，而C++中只有0是。

nullptr

C++中NULL扩展为常数0，这是int型。因此如果有两个重载函数，其中一个参数为int型，另一个为char*型。

当调用该函数时传递的参数是NULL，而我们的本意是调用char*类型对应的重载版本，这时会出现我们不想要的结果，调用了 int 重载版本。

而关键字nullptr解决了这种二义性，它是一种空指针常量。

宏定义do{}while(0)

在定义形如此类的宏时

define FUNCTION(x) a()

使用

define FUNCTION(x) do{ a(); } while(0)

可以避免意向不到的语义上的错误，并且能够保证do{ }内的代码只被执行一次。

std::ios.sync_with_stdio(false)
